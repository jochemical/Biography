This project implements a Django-based wiki encyclopedia where users can view, add, and edit scientific content written in Markdown and rendered as HTML. Django models and migrations create a SQLite database to store the data. The frontend is built with HTML and CSS, and includes features such as a search function and random page selection. The focus lies on backend Django technologies, views, and dynamic HTML rendering.

*This web application is based on the 2025 course "CS-50's WEB PROGRAMMING WITH PYTHON AND JAVASCRIPT (CS-50W)" by ed-X Harvard-X
