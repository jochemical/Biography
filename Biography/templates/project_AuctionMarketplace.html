This Django web application features an auction marketplace where users can register, log in, and create listings by adding a title, description, category, image, and an initial bid. Other users can place bids on active auctions, and sellers have the ability to close their auctions. Additionally, users can post comments on listings, browse listings sorted by category, and maintain a personalized watchlist for items of interest. The backend utilizes Django with authentication decorators, models, migrations, timestamps, and SQLite as the database, while also leveraging the Django admin interface for managing model instances.

*This web application is based on the 2025 course "CS-50's WEB PROGRAMMING WITH PYTHON AND JAVASCRIPT (CS-50W)" by ed-X Harvard-X
