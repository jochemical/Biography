Social Network is a Single Page Application built with Django and JavaScript. Users can register, log in, post and edit messages, like or unlike posts, and follow or unfollow others. Posts and user data are stored in a SQLite database and displayed in reverse chronological order, divided across pages using Bootstrap pagination. Specific query string parameters in the URL are used for page navigation. The Following view displays posts from followed users, while the Profile view shows a user's profile and their posts. The main challenge was handling dynamic interactions using JavaScript within a Single Page Application. Fetch requests retrieve specific data from the backend and update the DOM without reloading the page, creating a fast and responsive user experience. This project allowed me to sharpen my JavaScript skills and learn how to optimize fetch requests to work more efficiently with backend data.

*This web application is based on the 2025 course "CS-50's WEB PROGRAMMING WITH PYTHON AND JAVASCRIPT (CS-50W)" by ed-X Harvard-X
