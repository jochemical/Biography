This Django web application was developed as a practice project, with a primary focus on design quality and user experience. It demonstrates a bilingual setup, allowing users to switch seamlessly between Dutch and English, and integrates SoundCloud audio tracks for interactive multimedia content. Key challenges included optimizing page load times despite Google Fonts, high-resolution images, and audio files, which was addressed with background overlays for gradual image loading. JavaScript was used to provide image enlargement functionality while using smaller preview images for an improved user experience. The application also features a comprehensive content management system, allowing staff/admin users to edit pages, images, text, and audio. The project remains in development, with ongoing improvements and feature additions.
