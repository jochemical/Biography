The Discussion Forum is a small web application built with Flask, where users can post individual messages to discuss issues related to a statement displayed at the top of the page. This project incorporates functionalities that offer valuable learning experiences and can be applied to many other projects.

Throughout this project, I enhanced my understanding of forms and POST methods and improved my skills in HTML, Jinja2, CSS, and Python (using Flask libraries). Most notably, I gained experience with reading from and writing to a MongoDB (JSON) database to store forum messages, including their timestamps.

*This project was based on the course 'Web Developer Bootcamp with Flask and Python' of Jose Salvatierra (Teclado).

Technologies used

  • Python
  • HTML
  • Jinja2
  • CSS
  • MongoDB
