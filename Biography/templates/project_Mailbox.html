In this project, I developed the frontend of a Django-based email client. Users can register, log in, view their inbox, read and reply to emails, compose new messages, and manage email status by archiving or unarchiving messages. The frontend uses JavaScript to dynamically update the interface without reloading the page. This is achieved through asynchronous fetch requests to the Django backend connected to an SQLite database.

*This web application is based on the 2025 course "CS-50's WEB PROGRAMMING WITH PYTHON AND JAVASCRIPT (CS-50W)" by ed-X Harvard-X
