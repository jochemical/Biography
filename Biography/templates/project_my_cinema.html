My Cinema is a webapplication in which users create their own personal movie watchlist and add all kind of information including the year of production, last-watched date, director, cast, tags, series, description, star-grading and the embedding of a Youtube-iframe. The user should first register by creating a personal account using an e-mail adress and a new password. With this account the user can login and see his personal list of movies. Movies are added by using the add-button at the lower right corner. After clicking this button, a movie can be added by using an input form. Extra information can be added or adjusted later on using the edit-link. CSS-variables enable to choose between a light or a dark mode.

After this project I learned a lot of new libraries and backend techniques, including app-configurations, dataclasses, classes to set up inputforms, Flaksforms, wtforms, wtform fields and validators, hashing passwords, decorators (wrappers), secret keys, store user information within the session cookies, flash messages and more. Also within the frontend I applied a lot of different new techniques like CSS-variables, Jinja2 pipe operators, BEM modifiers, HTML table structures, and a lot of Jinja2-macro's. This application will allow for diverse screensizes by adjusting to the screensize of the user. A MongoDB datebase is used to store user profiles and link them to their personal movies. It is created in Flask and uses coding languages as HTML (Jinja2), CSS, and Python. It was a great project to work on and I learned a lot of new coding techniques !

*This project was based on the course 'Web Developer Bootcamp with Flask and Python' of Jose Salvatierra (Teclado).
