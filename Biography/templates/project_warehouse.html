This Flask web application functions as a REST API for managing warehouse data. Users can register and log in to create stores, items, and tags, with all data stored in a SQLite database. The application operates within a Docker container.

Through this project, I gained experience in several key areas: working with data models using SQLAlchemy, performing data validation with Schemas, and managing data migrations with Alembic. I also learned to implement JWT access tokens for secure authentication. Additionally, I deepened my understanding of Docker, including its concepts such as images, volumes, containers, and Docker Compose. I further honed my skills using tools like Insomnia and DBeaver. Although this application lacks a frontend, the techniques and knowledge acquired are broadly applicable to various other projects.

*This web application is based on the 2024 course "REST APIs with Flask and Python" by Teclado, taught by Jose Salvatierra.

Technologies used

  • API
  • Python
  • Docker
  • Insomnia
  • SQLite
