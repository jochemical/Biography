This Flask web application serves as a REST API for managing warehouse data. Users can register and login to create stores, items, and tags, all stored in a SQLite database. The application runs within a Docker container. Through this project, I gained experience working with datamodels using SQLAlchemy, datavalidation with Schemas, and datamigrations using Alembic, and I learned to write JWT access-tokens for secure authentication. I also deepened my understanding of Docker, including images, volumes, containers and Docker Compose, and practiced using Insomnia and DBeaver. While this web application does not include a frontend, I acquired a wide range of techniques that are applicable to many other projects.

*This web application is based on the 2024 course "REST APIs with Flask and Python" by Teclado, taught by Jose Salvatierra.

Technologies used

  • API
  • Python
  • Docker
  • Insomnia
  • SQLite
