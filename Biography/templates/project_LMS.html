Full-stack web application using React and Django with a MySQL backend. Database designed in DrawSQL and implemented in MySQL Workbench using raw SQL, then migrated to Django with models and migrations. Django REST API connects the frontend to the MySQL server, with ongoing feature development.

Platform for students, teachers, and tutors to share educational materials, track progress and grades, take online tests, manage schedules with a planner, and more. Frontend built with React (react-router-dom, hooks, state management) and styled with MUI components (DataGrid, Modal, Box, Toolbar, sx styling). JavaScript features such as async/await and interactive modals enable dynamic, responsive interactions.

  • REST API
  • React (MUI)
  • MySQL
  • HTML & CSS
  • JS & JSX
