Fibonacci Game is an interactive game on a 50x50 grid where players strategically click cells to increase the values in the corresponding rows and columns. The objective is to create horizontal or vertical Fibonacci sequences of five or more cells. When a complete sequence is formed, the highest values are highlighted in yellow, the involved cells are temporarily stored and then reset to zero, and the player's score is increased. The game is implemented using Tkinter and CustomTkinter to provide a responsive and visually appealing GUI, including scrollable windows, interactive buttons, score counters, and hover effects. A second version of the game was later developed using PyGame to experiment with alternative rendering and interactivity.

During this project, I applied a logger to neatly record and monitor error messages, ensuring reliable runtime behavior. The code follows best programming practices, including clear naming conventions for variables, constants, functions, and classes, proper commenting, error handling, and use of object-oriented design with methods and inner classes. The algorithm efficiently detects Fibonacci sequences by evaluating only relevant rows and columns around the clicked cell, tracking sequence direction, and minimizing computational load. Cells and sequences are rendered using Canvas and Frames, providing a smooth and intuitive user experience. The codebase is structured to be scalable and maintainable, allowing easy extension or further development of the game.

Technologies used

  • Python
  • OOP
  • Tkinter GUI
  • CustomTkinter
  • PyGame
