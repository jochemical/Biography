This Django web application leverages OpenAI to provide code corrections and suggestions for personal code development. Users can input their code and have it fixed by OpenAI. Additionally, users can submit questions to generate code for specific purposes. The application requires users to select a programming language from a dropdown menu. To use the application, users must first register and log in. The "Past" page allows users to review code they previously fixed or generated. User data is stored in a relational SQLite database.

During this project, I deepened my knowledge of the Django framework and became more familiar with Bootstrap for creating responsive and visually appealing web designs quickly. The most enjoyable aspect of this project was learning how to integrate OpenAI into a web application. As AI becomes increasingly significant in web development, exploring how it can be used and integrated is both fascinating and valuable. Overall, it was an exciting and rewarding project!

*This project was based on the course 'Create a Code Writing ChatGPT A.I. Site With Django Python' of John Elder (Codemy).

Technologies used

  • Django
  • OpenAI
  • HTML
  • CSS
  • SQLite
