Weight Consultancy (Python/Wagtail + Vanilla JS) - Full-stack website and CMS built with Python and Wagtail, designed to be highly maintainable and fully manageable through the admin interface. The site functions as a complete CMS where most content can be configured through the admin panel, including images, StreamField blocks, and custom admin settings such as contact information. Content editors can easily structure pages using dynamic blocks, while the SQL-backed architecture ensures reliable data management. The project is currently ongoing and built with scalability and maintainability in mind.

The frontend is implemented with responsive HTML/CSS and Vanilla JavaScript, following the client's design tokens and style conventions to ensure visual consistency. JavaScript enhances the user experience with an expandable navigation menu and shadow overlay interactions. The interface emphasizes usability and clear UX/UI patterns aligned with the organization's guidelines, with extensive use of card-based layouts. Styling is organized using page- and component-specific CSS with BEM modifiers, creating a scalable architecture. The website also includes a structured contact page with an integrated Google Maps location, contact form, and social media links.
